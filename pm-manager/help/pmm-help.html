PM Manager Addon - 1.0.0 © 2017
Manage Users and Block Uninvited PM's
Description
This addon is designed to block users globally from sending you private messages by easily managing a simple whitelist implementation, which users you wish to allow to message you, while notifying both you and non-whitelisted users when such blocks occur.
You have at your disposal a flexible and easy to use context menu system and a couple of dialogs that allow you to add, remove or swap any user from your choice between a single temporary or a multiple permanent whitelist.
There is also an option setup dialog, you can configure a personalized message, which the non-whitelisted users will see, and to enable or disable notifications of blocked messages that are meant to keep you informed. In addtion you have a static and simple Backup and Restore system, so you can keep your settings or whitelist safe, this also allows a greater flexibility in managing your core whitelist entries, should you prefer.
Features
  • In channel context menu entries that control the PM Manager and the enables and disables the PM blocker function.
  • On user context menu entries that allow adding, removing, replacing or changing the target user from a temporary to/or permanent whitelist.
  • A dialog with user list display with control and access buttons.
    • Refreshable Channel Users and Whitelisted Users lists.
    • Enable/Disable notification of blocked messages.
    • Comprehensive tooltips in all controls that show hints of what controls do.
  • A static Backup and Restore functionality for permanent whitelist and main PM Manager configuration saving.
  • A comprehensive in active window feedback messaging, serving as visual confirmations.
  • Comprehensive support for keyboard access via Alt+"Key"
  • A comprehensive help file.
Usage
Once you have the PM Manager Addon installed successfully.
Via Context Menus
  • Right-click in any channel window of your choice to access the PM Manager.
  • Right-click in any channel window of your choice to start or stop the PM Blocker.
  • To add a user: Select any user and via context menu and Add $user to Whitelist or Add $user As Temporary User
  • To remove a user: Select the target user and via context menu select Remove $user from Whitelist or Remove $user As Temporary User
  • To replace a temporary user: Select a any target user and via context menu select Replace User: $user with User: $user As Temporary User
  • To remove a temporary user making it permanent: Select the temporary user and via context menu select Remove $user As Temporary User and Add to Whitelist
  • Right-click in any channel window of your choice to access the PM Manager's Help. Yes, this file.
Via PM Manager Dialog
Once you have opened the main PM Manager window.
Type a temporary nick here... Enter any nickname of a user you wish to add to the temporary whitelist.
Channel Users Highlight any user under this list, then click the right pointing arrow button to add the user to the whitelist.
Whitelisted Users Highlight any user under this list, then click the left pointing arrow button to remove the user from this whitelist.
Blocker ON/OFF Click button to turn the PM blocker function ON ( block those pesky PM's ) or OFF ( Allow all PM's )
Refresh With the PM Manager open, you can move to any channel on any server.
Click the Refresh button and the Channel Users list will be re-populated with those channel's users.
Options Click this button open the Options Setup Dialog.
Exit Click this button to close the main PM Manager dialog.
Notes: The whitelist is auto-saved as users are added to or removed from this list.
The temporary user whitelist, can only hold one user at any one time.
The Channel Users and Whitelisted Users lists, only one user can be added/removed per any one selection/click.
Via Options Setup Dialog
Once you have opened the Options Setup window.
Custom notification setup: You can personalize the notice message that is sent out on blocked messages.
Notification of blocked queries is ON: (Default Checked) Opens a notification window when a message is blocked.
If you prefer not seeing any notifications about your blocked messages, just uncheck this box.
Add Click this button to add the message that you typed under Custom notification setup:
Edit Click this button to edit the message that you previously added/saved under Custom notification setup:
Reset Click this button to reset the currently saved message to default. (Automated Message: Uninvited PM's are not being accepted.)
Save Click this button to save your message or changes that you typed/edited under Custom notification setup:
Help Click this button to access the built in help and documentation.
The Backup and Restore Options
Backup Click this button to backup your current Whitelisted Users list and the PM Manager Setup configuration.
Restore Click this button to restore your current Whitelisted Users list and the PM Manager Setup configuration.
Close Click this button to cancel any unsaved changes and returns you to the main PM Manager dialog.
Notes: Most buttons are conditionally enabled/disabled in relation to their usage. Its important to note what you can do,
The Backup button, is disabled until a initial backup has been made. Thereafter its always ON to enable you to refresh your current backed-up files.
The Restore button is disabled until the Whitelisted Users list has been modified, and you can use the Restore function to reset your list to a previous backup.
Because the Restore rule used to disable the button, you would be able to restore your Whitelisted Users list in case the main is empty or has less or more entries than the backup.
Temporary Users are not backed-up at this time.
If you're interested in ensuring your Whitelist and Configuration is kept safe and up-to-date, do remember to make backups regularly. ;)
