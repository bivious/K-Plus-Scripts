#MOVIEGODS
  [MG] XDCC Queue II      V1.1.0   Addon   2017
  Description:

   A basic sequential download queue with some very unique features.


  Features:
  1. Featuring GUI Interfaces for several functions.
  2. Simultaneous downloads from 1 to 4.
  3. Control Regional Bots when using the 'sendfile' method.
  4. Control Passive Bots in different ways
  5. Set a delayed time to start your downloads. (24 Hour period)
  6. Catch stalled downloads to keep the queue running.
  7. User can select a minimum download speed for the queue run.
  8. If speed drops below minimum setting, file is removed from queue but will continue to download.
  9. User has complete control of this feature, it can be turned ON or OFF.
  10. User selected speed from 25 KiB/s to 500 KiB/s in steps of 25 KiB/s.
  11. Predefined KVIrc Accel Keys that can be used in the Control Center.

  Usage:
  • Use the GUI Configuration Panel to setup the loader.
  • Enable the GUI Queue Manager and use the Search button to begin.
  • If the Search button is greyed out. Turn the QLoader on.
  • Enter your search parameters in the search box widget.
  • Open the window labeled [MG] XDCC Queue Loader.
  • Right Click on the [A2Q] button when you see it highlighted.
  • Repeat the process to add additional files.

   Check the listing to verify you have the files you want.
   Now you are ready to start downloading or set a delayed time to start.


  XDCC Queue Loader Reccommendations:

  While this feature allows a simple Right-Click to add files.
  Please note this script is based on a basic search setup;

  | %GETSx | %SIZE | %RLS | /msg %NICK xdcc %SSEND %PNUM | %STATUS %OFFLINE

  *** And may only work properly with this setups ***
