  GUI Queue Manager and Controls
  Description: 

 The GUI Queue Manager once opened is an interactive interface.
 When a file is added to the queue the list will be updated.
 When a file is triggered for download it will be deleted from the list.

  GUI Queue Manager Buttons:
  • [ Remove Queued Item ]
    • Will remove an item from the queue listing.
    • Select an item with the mouse or arrow keys.
    • Click on the button or use Alt+R to remove the selected item.
    • Button will be greyed-out when the queue is empty.
  • [ Reset Queue ]
    • Will reset the queue removing all item(s) from the queue listing.
    • Requires a confirmation before proceeding for safety.
    • Click on the button or use Alt+Q to reset the queued item(s).
    • Button will be greyed-out when the queue is empty.
  • [ Begin Run ]
    • Starts the download process of the queued file(s).
    • Click on the button or use Alt+B to start downloading.
    • Button will be greyed-out while the queue is running or empty.
  • [ Pause / Resume ]
    • This is a combo button and acts as a toggle.
    • Pause the current queue run, the button changes to Resume.
    • Click on the button or Alt+P to pause the current run.
    • Once paused, click on the button or use Alt+u to resume the current run.
    • Button will be greyed-out when the queue is empty.
  • [ Delay ]
    • This is a combo button and acts as a toggle.
    • Set / Cancel a time delayed start, the icon will change depending on mode.
    • Click on the button or use Alt+D to either set or cancel the delay.
    • Button will be greyed-out while the queue is running or empty.
  • [ Search ]
    • Initiates a file search by a popup search widget.
    • Click the button or use Alt+S to activate.
    • You can enter exclusions like -720p,german before the filename.
    • Once your search parameters have been entered.
    • Click on the OK button or hit the Enter key to start the search.
    • Users that wish to use Reqular Expression may do so by checking the Enable REGEX Searches.
    • Click on the Cancel button to abort the search.
    • Button will be greyed-out when the queue is empty or if the QLoader is OFF.
  • [ Turn Qloader ON/OFF ]
    • This is a combo button and acts as a toggle.
    • The buttons icon and name will change depending on mode.
    • If you toggle the QLoader OFF, the Search button will grey-out.
    • When you toggle the QLoader ON, the Search button will be active.
    • Click on the button or use Alt+O or Alt+F to activate the toggle.
  • [ QInfo ]
    • Displays this widget for easy access to functions.
    • Can be minimized to the taskbar for easy access.
    • Click on the button or use Alt+I to activate.
  • [ Setup Queue II ]
    • Displays the configuration panel.
    • Changes the setting you want and Apply.
    • Or Cancel to abort any changes.
    • Click on the button or use Alt+e to activate.
  • [ NInfo ]
    • If you also have [MG] Announcer-Ninja Addon installed.
    • Displays the Ninja's widget for easy access to functions.
    • Can be minimized to the taskbar for easy access.
    • Click on the button or use Alt+N to activate.
  • [ Close Me ]
    • Exit the GUI Queue Manager.
    • Nothing more, nothing less.
    • Click on the button or use Alt+C to close the GUI.
