#MOVIEGODS
  [MG] XDCC Queue II      V1.1.0   Addon  2017
  XDCC Control Center - Accel Keys:
  • [ F2 ] - /mgq::xdccq cheats
    • Uses a 'KVIrc TabWidget' with multiple tabs.
    • Shows Info, Accel Keys, Commands and Queue Manager buttons.
    • Can be minimized to the taskbar for easy access.
  • [ F3 ] - /mgq::xdccq qmngr
    • Displays the GUI Queue Manager.
    • Contains all functions neccessary to manage the queue.
    • Can be minimized to the taskbar for easy access.
  • [ F4 ] - /mgq::xdccq start
    • If you have files queued, it begins the downloading process.
    • Executes immediately.
  • [ F5 ] - /mgq::xdccq search
    • Pops up the search widget if the QLoader is ON.
    • Executes immediately.
  • [ F6 ] - /mgq::xdccq qloader
    • Toggles the QLoader ON or OFF.
    • Executes immediately.
  • [ F7 ] - /mgq::xdccq setup
    • Displays the Configuration panel.
    • Executes immediately.
