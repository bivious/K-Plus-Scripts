#MOVIEGODS
  [MG] XDCC Queue II      V1.1.0   Addon  2017
  Manual Commands & Default Settings:
  XDCC Loader:
  • QLOADER
    • Toggles XDCC Queue Loader ON or OFF
    • Usage: /mgq::xdccq qloader
    • Default: OFF
  • SETUP
    • Displays QLoader Configuration Panel
    • Usage: /mgq::xdccq setup
    • Default: None

  XDCC Queue:
  • CHEATS
    • Uses a 'KVIrc TabWidget' with multiple tabs.
    • Usage: /mgq::cheats
    • Shows Info, Accel Keys, Manual Commands and Queue Manager Details.
    • Can be minimized to the taskbar for easy access.
    • Default: None
  • START
    • Starts the Q-Run now.
    • Usage: /mgq::xdccq start
    • Default: None
  • SEARCH or !S
    • Initiate file search within the Control Center.
    • Enter your search parameters in the dialogue popup window.
    • Usage: /mgq::xdccq search or !s
    • Default: None

  To Remove [MG] XDCC Queue II:
  • UNINSTALL
    • Initiate the Manage Addon to completely remove this script.
    • User will be prompted to proceed or abort the uninstaller.
    • Usage: /mgq::xdccq uninstall
    • Default: None
